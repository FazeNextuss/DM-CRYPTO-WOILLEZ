
# TP : Cryptographie

On souhaite crypter un texte en décalant d’un certain rang les lettres.

On pourra se restreindre aux textes ne comportant que des lettres minuscules et des espaces.




## Usage

1- Run the script

2- In the console :
```py
decaler("a", 1)
# Return 'b'

crypter("pierre", 1)
# Return 'qjfssf'
```
